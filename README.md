In the landing page, there's a guide on how the user will log in and use the system for the user side.

Home - provides an overview of the Family Diagnostic Center (FDC), which includes an online appointment booking system.
Key Features of the Family Diagnostic Center’s system, designed to make healthcare management easier and more efficient for both patients and staff: 
HIPAA Compliant – Ensures patient data is protected and follows healthcare privacy laws.
Real-time Updates – Keeps all information instantly synchronized across devices.
Mobile Access – Allows users to view patient records and appointments anytime, anywhere.
Easy Integration – Connects smoothly with other existing healthcare systems.
Services Offer - patients can view all the services offered by FDC, including ultrasound, 2D echo, vascular studies, and DNA testing.
About FDC - the patients can see all the information needed about FDC so that they become aware of where it is located and where it started.
